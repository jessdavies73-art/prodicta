# Education shell (Phase 2)

Reserved for teachers, teaching assistants, SEN specialists, nursery practitioners, tutors, and education management roles.

The education Workspace differs from Office: lesson planning, behaviour incident reporting, parent communications, safeguarding, SEN paperwork, exam season pressures, leadership team escalations. Block library and shell layout will be designed in Phase 2.

Currently the role-profile-detector classifies teaching and education-support roles as `shell_family = 'education'`. Phase 1 does not generate a `workspace_scenario` for these roles; the assess flow falls back to the legacy WorkspacePage.
