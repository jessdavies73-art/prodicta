# Field operations shell (Phase 3)

Reserved for hospitality managers, retail managers, security managers, hotel managers, restaurant managers, sports/leisure managers, real estate agents, creative leads, and pharma reps.

The field-ops Workspace differs from Office: floor walks, shift coverage, stock and rota issues, customer-facing incidents, on-the-move multi-channel communication, supplier failures, journey planning, and regulatory inspections. Block library and shell layout will be designed in Phase 3.

Currently the role-profile-detector classifies these roles as `shell_family = 'field_ops'`. Phase 1 does not generate a `workspace_scenario` for these roles; the assess flow falls back to the legacy WorkspacePage.
